OpenJDK Project Builds

What are these binaries?

These binaries are built by Red Hat on their infrastructure on behalf of the OpenJDK jdk8u and jdk11u projects. The binaries are created from the unmodified source code at OpenJDK. Although no formal support agreement is provided, please report any bugs you may find to https://bugs.java.com/. If you are looking for the AdoptOpenJDK supported binaries then please find them here.
